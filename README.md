# anrhouses-case studies

This repository contains the data and codes developed within the ANR-funded [HOUSES](https://anrhouses.github.io/).
- [SyntheticCases](./SyntheticCases) contains the data and R scripts to upload the synthetic cases used by [Rohmer 2025](https://www.sciencedirect.com/science/article/pii/S1574954125004686) and [Labourg et al., 2024](https://hal.science/hal-04886105v1).
- [CasEau](./CasEau) contains the data and R scripts to upload the case on water quality used in [Perchtold et al., 2026](https://meetingorganizer.copernicus.org/EGU26/EGU26-8056.html).
- [Toulouse](./Toulouse) contains the data for the Toulouse soil pollution test case used by [Rohmer et al., 2024](https://soil.copernicus.org/articles/10/679/2024/).
- [Dune](./Dune) contains the data for the intercomparison exercise used by [Rohmer et al., 2025](https://geostat25.sciencesconf.org/662476).
- [Modis](./Modis) contains the data for the intercomparison exercise used by [Rohmer et al., 2025](https://geostat25.sciencesconf.org/662476).
